# MyMind 备忘录 — 设计文档

## 1. 产品概述

MyMind 是一款微信小程序备忘录工具，采用**无限画布**交互范式，用户可以在一个可自由平移和缩放的画布上放置文字纸条、图片、语音条三种类型的备忘元素，实现灵感收集、思维整理和快速记录。

### 核心理念
- **无限画布**：借鉴 Miro/便签墙概念，无固定布局，自由放置
- **多模态输入**：文字、图片、语音三种内容类型
- **操作极简**：底部工具栏一键添加，直接拖拽定位
- **清新简约**：薄荷白底色 + 三色便签，视觉舒适不干扰

## 2. 用户画像

| 特征 | 描述 |
|------|------|
| 年龄 | 18-35 岁 |
| 场景 | 灵感速记、会议记录、读书笔记、待办管理 |
| 痛点 | 传统备忘录线性列表不够自由，无法直观组织信息 |
| 期望 | 快速记录、自由排列、视觉化组织 |

## 4. 交互设计

### 4.1 画布操作
| 操作 | 触发方式 | 行为 |
|------|---------|------|
| 画布平移 | 空白区域单指拖动 | 整个画布跟随手指平移 |
| 双指缩放 | 双指捏合/展开 | 以双指中心点缩放（0.25x ~ 3x） |
| 双击缩放 | 双击空白区域 | 在 1x ↔ 1.5x 切换 |
| 适配全部 | 点击"适配"按钮 | 缩放并平移使所有元素可见 |

### 4.2 元素操作
| 操作 | 触发方式 | 行为 |
|------|---------|------|
| 拖拽移动 | 按住元素移动 > 10px | 元素跟随手指，显示 Ghost 轮廓，松手保存 |
| 选中 | 短按元素 | 底部弹出选中工具栏 |
| 长按菜单 | 长按元素 500ms | 触觉反馈 + 弹出上下文菜单 |
| 缩放 | 选中后拖拽右下角手柄 | 调整元素尺寸（文字仅宽度，图片等比） |
| 双击编辑 | 双击文字纸条 | 进入编辑模式，显示 textarea |

### 4.3 元素交互明细
| 元素类型 | 选中行为 | 编辑行为 | 删除方式 |
|---------|---------|---------|---------|
| 文字纸条 | 显示选中框 + 缩放手柄 | 双击进入编辑，失焦自动保存 | × 按钮 / 选中栏 / 长按菜单 |
| 图片纸条 | 显示选中框 + 缩放手柄 | 仅标题可编辑 | × 按钮 / 选中栏 / 长按菜单 |
| 语音条 | 显示选中框 | 仅标题可编辑，点击播放/暂停 | × 按钮 / 选中栏 / 长按菜单 |

### 4.4 录音交互
- 点击底部 🎤 → 开始录音（按钮变红 + 脉冲动画）
- 再点击 ⏹ → 停止录音，自动上传并创建语音条元素

### 4.5 锁定
- 锁定后元素不可拖拽、缩放、编辑内容
- 锁定元素仍可选中、长按弹出菜单
- 头部显示锁图标，选中框变灰色

| 图片纸条 | 底 #EFF6FF / 头 #BDE0FE | 淡蓝相框 |
| 语音纸条 | 底 #F3F0FF / 头 #E4DCFF | 淡紫胶囊 |
| Ghost 轮廓 | rgba(108,99,255,0.4) 虚线 | 拖拽时原位置占位 |
| 选中框 | #6C63FF 2rpx 实线 | 锁定时 #95A5A6 |
| 缩放手柄 | #6C63FF 28rpx 方块 | 选中时右下角 |
| 删除按钮 | × 图标 56rpx | 头部右侧，按压态淡红 |
| 文字主色 | #2D2D3F | 深灰黑 |

| 元素 | 圆角 | 内边距 | 阴影 |

| 选中栏 | 36rpx | 0 8rpx | 0 4rpx 20rpx rgba(108,99,255,0.12) |
| 上下文菜单 | 20rpx | 8rpx 0 | 0 8rpx 40rpx rgba(0,0,0,0.12) |
| 缩放手柄 | 6rpx | - | 0 2rpx 8rpx rgba(108,99,255,0.3) |

### 5.4 动效
| 场景 | 效果 | 时长 |
|------|------|------|
| 纸条出现 | scale(0.8) → scale(1) | 200ms ease-out |
| 拖拽中 | scale(1.05) + opacity 0.85 | 实时 |
| 选中栏出现 | translateY(20rpx) + fadeIn | 200ms ease-out |
| 上下文菜单 | scale(0.85) + fadeIn | 180ms ease-out |
| 画布平移 | 跟随手指 | 无动画 |
| 录音脉冲 | scale(1) → scale(1.15) → scale(1) | 1000ms infinite |

## 6. 数据模型

### users 集合
| 字段 | 类型 | 说明 |
|------|------|------|
| _openid | String | 微信 openid（自动） |
| nickName | String | 用户昵称 |
| avatarUrl | String | 头像 URL |
| createdAt | Date | 注册时间 |
| updatedAt | Date | 最后更新时间 |

### notes 集合
| 字段 | 类型 | 说明 |
|------|------|------|
| _id | String | 文档 ID（自动） |
| _openid | String | 所属用户 openid（自动权限） |
| type | String | "text" / "image" / "audio" |
| x | Number | 画布 X 坐标 |
| y | Number | 画布 Y 坐标 |
| title | String | 标题 |
| content | String | 文字内容（仅 text 类型） |
| src | String | 文件 fileID（仅 image/audio 类型） |
| duration | Number | 音频时长秒数（仅 audio 类型） |
| width | Number | 元素宽度 |
| height | Number | 元素高度（仅 image 类型） |
| zIndex | Number | 图层顺序 |
| locked | Boolean | 是否锁定 |
| createdAt | Date | 创建时间 |
| updatedAt | Date | 更新时间 |
