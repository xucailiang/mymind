# MyMind — 无限画布备忘录

多模态备忘录微信小程序，支持在无限画布上自由放置文字纸条、图片、语音条，并可拖拽、缩放、锁定、排序。

<img src="image.png" alt="MyMind 页面截图" width="375">

## 功能

### 画布
- **无限画布** — 单指平移，画布无限延伸
- **双指缩放** — 双指捏合缩放，0.25x ~ 3x
- **双击缩放** — 双击空白区域在 1x ↔ 1.5x 切换
- **右侧面板** — 撤销/重做 + 缩放控件 + 适配按钮

### 元素操作
- **直接拖拽** — 手指按住元素移动即可拖拽，松手自动保存位置
- **短按选中** — 点击元素选中，底部弹出选中工具栏
- **长按菜单** — 长按 500ms 弹出上下文菜单（复制/删除/锁定/置顶/置底）
- **缩放手柄** — 选中后右下角出现缩放手柄，拖拽调整大小
- **锁定/解锁** — 锁定后元素不可拖拽、缩放、编辑

### 三种元素
| 类型 | 创建 | 交互 |
|------|------|------|
| **文字纸条** 📝 | 底部工具栏 → 文字 | 双击进入编辑，支持标题+正文 |
| **图片** 📷 | 底部工具栏 → 图片 | 从相册/相机选择，点击预览大图 |
| **语音条** 🎤 | 底部工具栏 → 录音 | 点击播放/暂停，波形动画 |

### 删除方式
- 元素头部 × 按钮
- 选中后底部工具栏"删除"
- 长按菜单"删除"

### 其他
- **撤销/重做** — 最多 50 步历史记录
- **图层排序** — 置顶/置底
- **复制元素** — 偏移 30px 创建副本
- **用户登录** — 微信授权登录，数据云端隔离
- **云存储** — 图片和音频自动上传，换设备不丢失

## 技术栈

微信小程序原生框架 + 微信云开发（云函数 + 云数据库 + 云存储）

## 部署步骤

### 1. 准备工作

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 1.02.19+
- 微信小程序账号（[注册](https://mp.weixin.qq.com/)）
- 基础库要求：2.10.0+

### 2. 导入项目

```
微信开发者工具 → 导入项目 → 选择 Mymind 目录
```

在 `project.config.json` 中填写你的小程序 `appid`。

### 3. 开通云开发

```
开发者工具顶部 → 云开发 → 开通
```

开通后记下你的**环境 ID**，填入 `app.js`：

```js
wx.cloud.init({
  env: '你的环境ID',  // 替换这行
  traceUser: true,
});
```

### 4. 部署云函数

在开发者工具中，右键以下目录分别选择 **"上传并部署：云端安装依赖"**：

- `cloudfunctions/login/`
- `cloudfunctions/init/`

### 5. 设置数据库权限

打开**云开发控制台 → 数据库**：

| 集合 | 权限规则 |
|------|---------|
| `users` | 所有用户可读，仅创建者可写 |
| `notes` | 仅创建者可读写 |

> 集合会在首次打开登录页时自动创建（通过 `init` 云函数），但权限需要手动设置一次。

### 6. 运行

编译运行，首次打开会自动：
1. 调用 `init` 云函数创建数据库集合
2. 弹出微信授权登录
3. 进入画布页面

```
├── sitemap.json           # 小程序索引配置

│   └── audio-note/        # 语音条组件

│   ├── login/             # 登录云函数
│   └── init/              # 数据库初始化云函数
├── utils/

└── docs/
    ├── design.md          # 设计文档
    └── dev.md             # 开发文档
```
